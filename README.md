# Saharan-dust-impact-on-last-minute-hotel-bookings
🏝️ Data Engineering study debunking the impact of Saharan Dust (Calima) on Gran Canaria hotel bookings. Features an automated ETL pipeline (AEMET API + ISTAC data) and dynamic anomaly detection. Statistical results (r=0.268) prove demand is inelastic, providing data-backed evidence to protect ADR and optimize ad spend during weather events.
